## 动力源

#### 史莱姆笼 Slime Cage 提供动能

多种加工设备的动力来源之一。

需要用肉类喂食史莱姆，可以用漏斗自动喂食。

#### 岩浆怪笼 Magma Cube Cage 提供加热

多种加工设备的动力来源之一，在加工时提供热能。

需要用岩浆膏或煤炭饼喂食岩浆怪，可以用漏斗自动喂食。

## 其他方块

#### 史莱姆捕捉器 Slime Catcher

当小型史莱姆接触到捕捉器时，史莱姆将消失，捕捉器将变为史莱姆笼。

当小型岩浆怪接触到捕捉器时，岩浆怪将消失，捕捉器将变为岩浆怪笼。
